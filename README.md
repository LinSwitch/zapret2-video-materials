# Zapret2 - стратегии и примеры

Набор стратегий и конфигураций, использованных в видео по настройке Zapret2 для Linux и Windows 10.

📺 **[Смотреть видео-инструкцию на YouTube](https://www.youtube.com/watch?v=Vnq-9ji9wes)**

---

## Официальные репозитории разработчика
* 💻 **[Официальный репозиторий Zapret2 (bol-van)](https://github.com/bol-van/zapret2)** 
* 🪟 **[Репозиторий Zapret для Windows](https://github.com/bol-van/zapret-win-bundle)** 
---

⚠️ Важно: данные стратегии не являются универсальными и могут перестать работать после изменений DPI или особенностей фильтрации у конкретного провайдера.

## Содержимое репозитория

### Linux

* `config-from-video` — конфигурация, показанная в видео.
* `config-current` — моя текущая рабочая конфигурация.

### Windows

* `zapret2-from-video.cmd` — скрипт запуска со стратегией из видео для Zapret2.
* `zapret-first-gen.cmd` — скрипт запуска со стратегией из видео для Zapret.

### 🛠️ Scripts
* `parse_blockcheck2.py` — кроссплатформенный скрипт на Python для Linux и Windows.
* `parse_blockcheck2.ps1` — скрипт для Windows (PowerShell).
* `start_parse.bat` — **Запуск скрипта в один клик для Windows!** Автоматически обходит запреты системы на запуск PowerShell-скриптов. 
* `clean_summary.txt` — очищенный пример лога для тестов.

Скрипты по умолчанию ищут файл `blockcheck2.log` в той же папке, где находятся. Для python-скрипта можно вручную указать входной и выходной файлы:

```bash
python parse_blockcheck2.py input.file output.file
```

## FAQ

<details>
<summary><b>Чем Zapret2 лучше первой версии?</b></summary>

Если первая версия у вас работает и полностью устраивает — можно ничего не менять.

Для новых установок имеет смысл использовать Zapret2, так как автор прекратил развитие первой версии и сосредоточился на второй.

</details>

<details>
<summary><b>Как подобрать стратегию для своего провайдера?</b></summary>

После завершения работы blockcheck откройте log-файл и найдите раздел SUMMARY.

Ниже будет список потенциально рабочих стратегий. Обычно достаточно перейти к результатам curl_test_https_tls12 и начать проверку найденных вариантов на практике.

Универсальной стратегии для всех провайдеров не существует.

</details>

<details> <summary><b>Откуда взялись файлы со стратегиями для Windows? Нужно ли их скачивать отдельно?</b></summary>

Все необходимые файлы уже включены в официальный архив разработчика.
Вам не нужно ничего искать на стороне, достаточно отредактировать уже имеющиеся файлы под свою стратегию.
</details>


<details>
<summary><b>Windows Defender считает Zapret вирусом. Это нормально?</b></summary>

Да.

Об этом говорится как в видео, так и в репозитории разработчика. Утилита работает с сетевым трафиком на низком уровне, поэтому некоторые антивирусы могут выдавать ложноположительные срабатывания.

</details>

<details>
<summary><b>Будет ли видео по OpenWRT?</b></summary>

Пока не планируется.

У меня нет роутера с OpenWRT, поэтому я не могу протестировать конфигурации и показать рабочий вариант на практике.

</details>

<details>
<summary><b>Где искать свежие стратегии?</b></summary>

Рекомендую периодически заглядывать в разделы Issues и Discussions в репозиториях разработчиков.

Там пользователи часто публикуют свежие конфиги, результаты тестов и рабочие стратегии для разных провайдеров и типов DPI.

</details>
